# 👋 Welcome

Welcome to the **official GitHub Organization** of the **Information Technology Innovators Club (ITIC)** at **Chinhoyi University of Technology (CUT)**.

ITIC is a student-led technology community committed to developing the next generation of software engineers, cybersecurity specialists, AI researchers, cloud engineers, entrepreneurs, and technology leaders.

We believe that the best way to learn technology is by **building real-world solutions together.**
